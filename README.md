## Use Tailwind CSS

To set up Tailwind CSS in your project, follow these commands:
With Tailwind CSS CLI
```bash
npm init -y
npm install
npm install -D tailwindcss
npx tailwindcss init
```
## useing post cs
```bash
npm init -y
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init
npx tailwindcss init -p
```
